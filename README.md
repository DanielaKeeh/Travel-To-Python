## Habilidades Demostradas 

### Fundamentos de Python
- **Sintaxis básica**: Variables, tipos de datos, operadores
- **E/S básica**: `input()`, `print()`, f-strings
- **Conversión de tipos**: `int()`, `float()`, `str()`
- **Operadores**: Aritméticos, compuestos, comparación

### Control de Flujo
- **Condicionales**: `if`, `elif`, `else`
- **Bucles**: `while`, control con banderas booleanas
- **Lógica booleana**: `and`, `or`, condiciones compuestas

### Estructuras y Organización
- **Funciones**: Definición, parámetros, retorno de valores
- **Módulos**: Importación, organización por carpetas
- **Menús interactivos**: Sistemas anidados, navegación

### Proyecto Final: Kokomi Bakery
- **Diccionarios**: Estructura de datos para productos
- **Validaciones**: Stock, existencia de productos
- **Cálculos**: Precios, cantidades disponibles
- **Persistencia básica**: Base de datos en diccionario

## Cómo Ejecutar

1. **Clona o descarga** este repositorio
2. **Se ocupa tener Python 3.8+** instalado
3. **Ejecuta el archivo principal**: --> python main.py
